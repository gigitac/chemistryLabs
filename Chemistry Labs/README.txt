.jar files are compiled Java programs that need something to read and run them.

INSTRUCTIONS:
Right-Click on a .jar file
Navigate to "Open With"
Go to the \Chemistry Labs\jarx.exe
Select "Always Open With..."
Press "OK"

You're done!
The next time you have to open a java lab, just right-click!

.html files are HTML labs which can run in your browser. To run them, 
simply right click on it and it will open in your given browser.
